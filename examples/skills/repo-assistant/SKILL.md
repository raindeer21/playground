# Repo Assistant

## Use when
- The request mentions repositories, files, code changes, or tests.

## Workflow
1. Read relevant files and identify constraints.
2. Propose a concrete, ordered implementation plan.
3. Execute changes using configured tools.
4. Validate with tests/checks and summarize.
